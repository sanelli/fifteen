# fifteen
[15 Puzzle](https://en.wikipedia.org/wiki/15_Puzzle) game implemented in C++ and SDL.

## OSX Setup
- brew install sdl2
- brew install sdl2_image
- brew install sdl2_ttf
- cmake .
- cmake --build .
